背景和障碍物

添加背景

使用的背景图片是:

图片下载到资源目录后, 在GameScene.js文件中的preload函数中，预加载图片：


this.load.image('sky', 'assets/sky.png');

接着在 create 函数中，给图片添加到场景中


this.add.image(400, 300, 'sky')
...

*** 在create函数开始的时候添加最底层的图片精灵，默认情况下后添加的精灵会在最上层。

跳跃

目前的主角还不能跳跃。我们使用小键盘的 上 来控制主角跳跃:

在update函数中 已经有了 向左和向右的控制，添加向上也很简单.


if (this.cursors.up.isDown) {
  this.player.setVelocityY(-300);
}

如果只是这样 我们一直按上的话 玩家可以一直向上而不掉下来，所以要添加一个调整, 在添加一个玩家在地上的条件：

if (this.cursors.up.isDown && this.player.body.onFloor()) {
  this.player.setVelocityY(-300);
}

添加台阶

台阶或者障碍物也需要一些图片来表示。

下载资源到本地assets目录以后，通过preload进行预加载


this.load.image('ground', 'assets/platform.png')

由于我们这些障碍我都是静态，也不会移动和被销毁，我们可以通过使用staticGroup() 来创建他们

在create函数中加入


this.platforms = this.physics.add.staticGroup() // 具有物理属性，跟玩家可以碰撞而不至于穿过去
this.platforms.create(400, 568, 'ground').setScale(2).refreshBody()
this.platforms.create(600, 400, 'ground')
this.platforms.create(50, 250, 'ground')
this.platforms.create(750, 220, 'ground')

现在运行的时候，玩家还是穿越了障碍物， 所以还差一步：在create函数中


this.physics.add.collider(this.player, this.platforms)

如此这般，玩家可以跳来跳去了
