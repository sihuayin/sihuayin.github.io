使用phaser3开发游戏01

准备

什么是phaser3?

请移步，到官网查看一下. phaser html5游戏框架

node, nvm , npm

由于node的多版本并存，和比较快的更新版本速度，我们本地环境最好能支持多个版本，能方便切换当前使用版本。那么可以选择安装nvm。

mac：


curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.2/install.sh | bash

linux:


wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.2/install.sh | bash

window:

使用安装包来安装nvm-window

检查nvm是否安装正确


nvm --version  // window: nvm version

安装node


nvm install node
nvm use node

查看和检查安装


node --version
npm --version

服务器

虽然html5的游戏是浏览器端程序，但是加载图片，音频等资源时候，需要一台静态资源服务器支持。 phaser网站也给出了几种简单的方案。在这里 我没使用 parcel: 快速，零配置的Web应用程序捆绑包。跟webpack等很类似，大家可以使用以后做一个比较。它可以给我提供一个测试使用的简单服务器。


npm install -g parcel-bundler
parcel --version

初始化项目

通过 模板项目 来初始化我们的游戏. 进入到我们的游戏项目的根目录执行克隆


git clone https://github.com/ourcade/phaser3-parcel-template my_first_game //项目名
cd my_first_game
npm install

启动server，查看效果


npm run start

打开浏览器，输入 http://localhost:8000, 初始工作完成。
